nyaos scripts
=============

Description
-----------
### gal.lua ###
nyaosであふwの左ファイル窓ｎフォルダに移動する


### gar.lua ###
nyaosであふwの右ファイル窓ｎフォルダに移動する


Requirements
------------
あふwでオートメーションサーバを有効にしてください。


Settings
--------
* _nyaの例
~/.nya/にluaスクリプトがある場合

    alias gar      source ~\.nya\gar.lua

    alias gal      source ~\.nya\gal.lua

Usage
-----
* あふwの左ファイル窓のフォルダに移動

    nyaos> gal

* あふwの右ファイル窓のフォルダに移動

    nyaos> gar
